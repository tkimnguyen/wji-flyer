# WJI Campaign Flyer (Static Site)

This is a simple static website suitable for **GitHub Pages**.

## Deploy on GitHub Pages
1. Create a new repo on GitHub (e.g., `wji-flyer`).
2. Upload the contents of this folder to the repo (or push via git).
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` (or `master`) / Folder: `/ (root)`
4. Save. Your site will appear at `https://<username>.github.io/<repo>/`.

## Files
- `index.html` — main page
- `assets/styles.css` — styling
- `assets/print.html` — printer-friendly version (also accessible from the main page)

## Customize
- Replace links, add a logo/photo, or adjust colors in `assets/styles.css`.
